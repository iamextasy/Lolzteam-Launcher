# Безопасность

## Как обрабатываются ваши данные

- **Данные аккаунтов нигде не сохраняются на диск.** Логины, пароли, 2FA-секреты
  и cookie запрашиваются у API lzt.market в момент каждого входа, используются
  для авторизации и сразу отбрасываются.
- **Нет телеметрии.** Лаунчер не отправляет ваши данные никуда, кроме официального
  API lzt.market и сервиса, в который вы входите.
- Локальные секреты (например, ваш API-токен) хранятся в зашифрованном хранилище
  ОС через Electron `safeStorage`, а не в открытых файлах.

## Защита

- Включены `contextIsolation` и `sandbox`; `nodeIntegration` в рендерере отключён.
- Строгая Content-Security-Policy ограничивает источники сети и скриптов.
- У рендерера нет прямого доступа к Node.js или файловой системе — все
  привилегированные действия проходят через типизированный IPC-мост.

## Сообщить об уязвимости

Нашли проблему с безопасностью? Пожалуйста, **не** создавайте публичный issue.
Сообщите приватно через [GitHub Security Advisories](https://github.com/iamextasy/Lolzteam-Launcher/security/advisories/new).
